# Installation et commandes de base de GIT

Créer un dossier « GIT »
clic droit puis « Afficher d’autres options » puis « Open git here » 

  
Récupérer le projet -> « Code » puis copier l’URL 
Dans GIT, taper « git clone (URL) »

git status 
git commit -a -m 
git push
